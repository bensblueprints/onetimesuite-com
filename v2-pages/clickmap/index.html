FAQ

Fair questions

Is it really free on GitHub?

Yes — Clickmap is MIT-licensed at github.com/bensblueprints/heatmap-session-recorder-mvp. The $49 buys the packaged Windows installer and updates; building from source is free.

Does it record what my users type?

No. The tracker has no code path that reads input values, textarea content or keystrokes — only element positions and short CSS selectors. It is physically incapable of recording typed input.

Is this a subscription in disguise?

No. $49 once, no renewal, no per-session pricing. Data never leaves your server, so you stay the sole controller.
