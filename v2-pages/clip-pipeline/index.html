FAQ

Fair questions

Do I need API keys?

A free Groq API key covers transcription and script rewrite. ElevenLabs is optional — there's a free local cloned voice built in for re-voicing with zero API calls.

Can it process a whole creator's page, not just one video?

Yes — paste a profile or page URL to scan and bulk-download every video, then run the whole batch through the pipeline unattended.

What happens to the original captions?

Removed. Four modes are available, from a fast blur to a scene-rebuild mode that regenerates each scene from a clean background so the original captions never exist in the output at all.

Is the source code available?

Yes — Clip Pipeline is MIT-licensed at github.com/bensblueprints/clip-pipeline. Buying the $49 package gets you the built installer; building from source is free.

Does it work for long-form-to-shorts too?

Yes — the YouTube Clipper mode turns one long video into multiple short, viral-scored vertical clips with captions burned in.
