FAQ

Fair questions

What's the difference between WholeTeam Admin and Member?

Admin is for the owner or manager — install it on one machine. It hosts the org chart, chat, files and time clock for everyone else, and it's the only place salary data ever lives. Member is for employees — it connects to the Admin install for chat, the file library, self-service profile and PIN clock-in.

Do I need a license per employee?

No — WholeTeam Admin is a one-time $19 purchase for the host machine, and employee Member installs are covered under the same purchase. There's no per-seat subscription.

Does anything ever touch the cloud?

No. Chats, files and hours stay on your own network. Remote employees connect back to the Admin machine over Tailscale — a private, encrypted overlay network — not through a public cloud service.

Is the activity tracking spyware?

No — there are no screenshots, no keylogging, no webcam. The activity % is idle-based only: it measures whether the machine was in use while clocked in, and it's visible to your admin, not to a third party.

Wasn't this called Orgtree?

Yes — WholeTeam is Orgtree grown up. The org chart builder is still here, now alongside team chat, department file libraries and the PIN time clock. Same one-time price.

Is the source code available?

Yes — MIT source at github.com/bensblueprints/wholeteam-mvp. The $19 buys the packaged Admin + Member installers and updates.
