## What we're doing here

Software went from something you *bought* to something you *rent*. A screen
recorder is $15/month. A form builder is $29/month. A status page is
$79/month. Stack ten of those and you're paying thousands a year, forever,
for tools that haven't fundamentally changed in a decade.

OneTime Suite is the counter-move: **102 apps that replace the most common
SaaS subscriptions**, each sold for a one-time price. Desktop apps run 100%
on your machine; web apps are self-hosted on a $5 VPS you control. No
accounts with us, no telemetry, no renewal dates, no lock-in.

## How the repos work: MVP (public) vs Full (private)

Every app in this suite exists in two editions:

- **`<app>-mvp` — public, MIT-licensed.** The bare-bones core of the app,
  free to read, build and run yourself, forever. These repos are
  intentionally minimal and are **not** where active development happens.
- **Full version — private.** The feature-rich edition that keeps getting
  updates: extra features, polish, signed installers, 1-click setup and
  ongoing support. This is what you get when you buy on
  [onetimesuite.com](https://onetimesuite.com) — still a one-time price.

If you like an MVP, the full version is the same app with the rough edges
sanded off and the power features added. That's the deal: the free MVP
proves the tool is real; the paid version is the one you'd actually want to
live in.

## About this repo (the website)

This repo is the static site generator for onetimesuite.com.

### Build

```
node build.js                 # generates everything into public/
node scripts/check-links.js   # verifies every internal link resolves
```

No npm dependencies — Node built-ins only.

### Source data

- `src/products.js` + `src/products-51-100.js` — the app catalog (data for the /software pages)
- `src/extra-products.js` — BloomRecorder + WisperTalk
- `src/posts-1.js` … `posts-6.js` — comparison posts
- `content/seo/*.md` — hand-written SEO posts
- `assets/shots/*.png` — real app screenshots
- `src/site.css` — the design system

### Deploy

```
docker build -t onetimesuite .
docker run -p 80:80 onetimesuite
```

Multi-stage Dockerfile: node:22-alpine runs `build.js`, nginx:1.27-alpine
serves `public/` with clean-URL `try_files`.
